# Customer Segmentation Analysis

## Overview
This project applies unsupervised machine learning techniques to segment customers based on behavioral and demographic patterns. The goal is to identify distinct customer groups that can inform targeted marketing strategies and improve campaign effectiveness.

## Dataset
Marketing campaign dataset containing customer demographics, purchasing behavior, and campaign response history.

## Approach
- Performed data cleaning and feature engineering to prepare customer-level attributes
- Conducted exploratory data analysis to understand distributions and relationships across features
- Applied clustering algorithms to group customers into meaningful segments
- Analyzed and profiled each segment to derive actionable marketing insights

## Tools and Technologies
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn, Google Colab

## Key Outcomes
- Identified distinct customer segments with unique purchasing behaviors and demographic profiles
- Provided segment-level recommendations for targeted marketing strategies
- Visualized cluster characteristics to communicate findings clearly
